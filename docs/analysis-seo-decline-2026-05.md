# SEO 流入急減 包括分析レポート

**作成日**: 2026-05-08
**対象期間**: 2024-12-24 〜 2026-05-03 (約 16 ヶ月)
**対象サイト**: ku-bell.com
**データソース**: Google Search Console API（Search Analytics API、過去 16 ヶ月の日次×ページ×クエリ全データを backfill）

---

## エグゼクティブサマリ

過去 16 ヶ月で **検索流入クリックが 73% 減** (47,965 → 12,830 / 月)。

決定的特徴：
1. **順位は逆に改善** (平均 7.0 → 5.5 位)
2. **インプレッションも増加** (798K → 1,242K / 月)
3. **にもかかわらずクリックが激減**

→ **Google AI Overviews (SGE) の影響**が主因と判定。
特に「比較」「危険」「やめた」「方法」など**情報系・質問系のクエリでクリック消失**が顕著。
ブランド名検索（クーベル）は耐性あり（横ばい）、純粋なカテゴリ・ハウツー流入が壊滅。

## 7. 真因の確証：AI Overviews (SGE) 影響

### 決定的証拠 6 点

1. **平均順位は逆に改善**（7.0 → 5.5 ＝ より上位表示）
2. **Impressions は維持〜増加**（798K → 1.24M ＝ Google には認識されている）
3. **Top 3 CTR が 10.96% → 1.56%**（一桁順位なのにクリックされてない）
4. **情報系・質問系クエリが集中砲火**（「危険」「比較」「方法」「やめた」）
5. **ブランド固有名は耐性**（指名検索は AIO に置換されにくい）
6. **2026-02 の崖**が日本市場での Google AI Overviews 本格展開タイミングと一致

### この症状の業界共通呼称

- **"Zero-Click Search"** （ゼロクリック検索）
- **"AI Overviews Cannibalization"**（AI Overviews による流入侵食）
- **"E-E-A-T 重要性の上昇"**（独自体験・専門性が無いコンテンツが弾かれる）

業界レポートでも同様の傾向：情報系メディアで 30-90% の流入減少報告が多発。

## 9. データソース・分析手法

### データソース

- **Google Search Console Search Analytics API** (over 16 months)
- **対象テーブル**: `campwill-ec.searchconsole.sc_history`
- **取得粒度**: 日付 × ページ (URL) × クエリ
- **総レコード数**: 7,088,233 行
- **対象期間**: 2024-12-24 〜 2026-05-03 (496 日)

### 分析手法

- 比較期間: **旧 = 2025-Q1** (2025-01-01 〜 2025-03-31) vs **新 = 2026-Q1** (2026-02-01 〜 2026-04-30)
- 順位帯分類: Top 3 / 4-10 位 / 11+ 位
- ブランド判定: クエリに `クーベル|kubell|ku-bell|クベル` を含む
- 重み付き平均順位: `SUM(position × impressions) / SUM(impressions)`

### 分析環境

- BigQuery (campwill-ec project)
- 分析用 SQL: `analysis/sql/seo-decline-*.sql` (再現可能)
- 自動更新: 翌朝 08:00 JST に Scheduled Query で更新（mart 層連携）

1. **総クリック数 / 総インプレッション数 / 全体 CTR**
